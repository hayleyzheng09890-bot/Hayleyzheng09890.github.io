# 图片资源目录

## 使用说明

请将您的图片放置在此目录中，然后在对应的 HTML 文件中引用。

## 建议的图片规格

### 个人照片
- 尺寸：360px × 360px
- 格式：JPG/PNG
- 建议：白色或浅色背景，正面半身照

### 证书图片
- 尺寸：800px × 600px (4:3)
- 格式：JPG/PNG
- 建议：高清晰度扫描件

### 实践照片
- 尺寸：1200px × 900px (4:3)
- 格式：JPG/PNG
- 建议：清晰展示活动场景

### 项目截图
- 尺寸：1600px × 900px (16:9)
- 格式：JPG/PNG
- 建议：产品界面或项目成果展示

### 二维码
- 尺寸：400px × 400px
- 格式：JPG/PNG
- 建议：清晰的微信二维码

## 命名规范

建议使用清晰的命名规则：

- `profile.jpg` - 个人照片
- `certificate-01.jpg` - 证书1
- `practice-01.jpg` - 实践照片1
- `project-01.jpg` - 项目截图1
- `wechat-qr.jpg` - 微信二维码

## 图片优化建议

- 压缩图片以减小文件大小
- 使用 WebP 格式可以获得更好的压缩效果
- 确保图片加载速度不影响用户体验
